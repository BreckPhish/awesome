Sorting rules
  • Table 1: Bussers, Dishwashers, Prep Cooks, Cooks, and all Three Daughters employees.
  • Three Daughters detection: employee name contains Three Daughters, job contains 3D, or location contains Three Daughters.
  • Table 1 names: first names only; Three Daughters rows get 3D after the name and sort first.
  • Table 1 hours: combined by displayed first name.
  • Table 2: Servers, Bartenders, Bar Lead, and Trainer. Trainee is excluded.
  • Table 2 layout: not combined; sorted by clock-in time and labeled as two independent FOH cash tip pools — the AM tip pool (clock-in before 2:00 PM) and the PM tip pool (clock-in 2:00 PM & after). Column 5 shows each row’s job title.
  • Always excluded: Online ordering profiles, Toast default profiles, Hosts, Managers (including Hourly Shift Manager), and Chefs never appear in Table 1, Table 2, or unmatched rows.
  • Column copy: each table has a button per column to copy just that column’s values. Table 2’s column buttons are split into the AM tip pool (before 2 PM) and PM tip pool (2 PM & after) so each column copies separately per pool.
  • Hours source: uses Payable Hours first, then falls back to Total Hours.
